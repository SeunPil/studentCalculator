## lombok 설정
1. build.gradle -> dependencies에 추가

```groovy
	compileOnly 'org.projectlombok:lombok:1.18.12'
annotationProcessor 'org.projectlombok:lombok:1.18.12'
```

2. 인텔리제이 SHIFT두번 눌러서 '플러그인' 검색 후 lombok 플러그인 설치
